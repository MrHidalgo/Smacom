Тесто

слоено-дрожжевое замороженное

Это тесто создано для настоящих сластён. Его можно использовать без дополнительной раскатки для максимального поднятия или раскатать в тонкий слой для приготовления тортов, пирогов, основ для пиццы. А удобная расфасовка теста листами (вместо рулона) позволяет размораживать лишь необходимое его количество.

Вкусной выпечки!

Cостав продукта
Мука пшеничная в/с, вода питьевая, маргарин для слоеного теста, ароматизатор натурально-идентичный «Масло-сливки», краситель натуральный β-каротин, лимонная кислота, дрожжи хлебопекарские, соль, сахар, хлебопекарский улучшитель, яичный порошок, ароматизатор идентичный натуральному «Ванилин».

Способ приготовления

Освободить тесто от внешней упаковки, разморозить при комнатной температуре (20-25 °C). При необходимости раскатать тесто до нужной толщины. Сформировать изделия, выложить на смазанный маслом противень. Для нарезания использовать острый нож или выемку, чтобы не смять края теста. Накрыть полиэтиленом во избежание обветривания. Поставить в теплое место (+25…+28 °C), дать подойти. Для придания готовым изделиям блеска и румяности, перед выпеканием смазать их поверхность яйцом. Разогреть духовку до 220-230 °C. Выпекать при температуре 200-220 °C до образования румяной корочки (мелкие изделия – 10-15 мин., крупные – 15-20 мин.).
